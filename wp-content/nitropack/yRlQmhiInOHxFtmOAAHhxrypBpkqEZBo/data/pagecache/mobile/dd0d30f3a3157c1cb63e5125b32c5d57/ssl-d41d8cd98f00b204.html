About us.

Founded in 2014 in Ukraine and inspired by France, Bato is an Agency of experts in web and mobile development.

We create websites of any type from scratch, ready to be used by our partners as well as provide any type of web development. Starting from 2014 we have accompanied more than 100 clients, accomplished 500+ projects!
